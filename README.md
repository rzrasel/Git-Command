# Git Command
**Git revision control as follows**
### Create a new local repository
```git-init
$ cd project directory
```
```git-init
$ git init
```
***Git will reply
Initialized empty Git repository in .git***
```git-init
$ git tag -a v1.2 9fceb02 -m "Message here"
$ push --tags origin master
```
Or you can follow:
```git-init
$ git tag -a v1.2 9fceb02 -m "Message here"
$ git push origin --tags
```
Or you can follow:
```git-init
$ git tag -a v1.2 9fceb02 -m "Message here"
$ git push --tags
```
